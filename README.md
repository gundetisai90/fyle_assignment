# GitHub Repos Viewer

This is a simple web application that allows you to enter a GitHub username, search for their repositories, and view the repositories with basic details, including the primary programming language used.

## How to Use

1. Open the `index.html` file in a web browser.

2. Enter a GitHub username in the input field and click the "Search" button.

3. The repositories associated with the entered username will be displayed, showing the repository name, description, and primary language.

4. Use the navigation buttons to browse through multiple pages of repositories.

5. Adjust the number of repositories per page using the "Update" button.

## Structure

- `index.html`: The main HTML file containing the structure of the web page.
- `style.css`: The CSS file with styles for the page layout.
- `jquery-3.5.1.min.js` and `bootstrap.min.js`: JavaScript libraries for handling DOM manipulation and providing additional styling, respectively.

## Dependencies

- Bootstrap (v4.5.2): [Bootstrap](https://getbootstrap.com/)
- jQuery (v3.5.1): [jQuery](https://jquery.com/)
